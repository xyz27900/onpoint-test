### Что это?
Тестовое задание для Onpoint

### Как посмотреть?
```shell
# Склонировать репозиторий
git clone https://github.com/xyz27900/onpoint-test.git

cd onpoint-test

# Установить зависимости
npm install

# Запустить
npm start
```
